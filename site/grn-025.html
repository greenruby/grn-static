#025 - jul 28th 2013

Edito

Hmm I didn't have much time this week for the gathering, as I'm busy changing job right now. But behind the scene we are working on preparing a tool for semi-automating the gathering process (well, Dale is the active guy there for now). Have a look at organews if you are curious and want to jump in.

Look

Examples of UI/UX, graphic performance, web design and flashy things.
Global land temperatures
A map of temperatures on the globe since 1900, nicely designed.
Soul reaper
Preview of a digital comic in html5 including sound. Great effects.

Use

Web applications, resources and tools, available for making our life easier or funnier.
Ruby Warrior
An adventure game involving ruby coding.
Bootstrap Hornbook
Collection of useful links when working with bootrstrap css.
Flynn
Well it's not ready yet but the idea is promising, an open-source Paas based on docker (2 days left to help sponsor it).
Practicing Ruby
A monthly publication about ruby. Note that they just opened their archives publicly.

Install

A selection of gems or applications updated during past week.
rails 3.2.14
Mainly a bugfix release, announcing that the next version will be the last one for the 3.2 family.
Bower 1.0.0
A total rewrite of bower and a big number release, may deserve to jump into.
Phonegap 3.0
Last week was announced a new version of that cross platform mobile html app development suite.
RubyMotion 2.5
Some interesting improvement in that release, making it more ruby-like.
ie8 ja shims
A set of tools for dom support in ie8.
Mix it UP
A really good css3/jQuery plugin for sorted lists of whatever with special effects.
MVCSS
A Sass-based css framework oriented towards OOCSS, BEM and SMACSS.
imagefill.js
The jQuery plugin for making images fill their containers (and be centered).
